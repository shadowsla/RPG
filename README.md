# Ultimate Tower RPG

An advanced action RPG tower crawler built with JavaScript and HTML5 Canvas.

Fight through dangerous towers, defeat massive bosses, unlock powerful subclasses, craft legendary gear, and become the strongest hunter.

---

# Features

## Combat System
- Fast-paced action combat
- Weapon mastery progression
- Combo attacks
- Critical hits
- Heavy attacks
- Air combat
- Ultimate abilities
- Parry system
- Dodge rolling
- Elemental damage

---

## Movement System
- Jumping
- Double jumping
- Sprinting
- Wall jumping
- Climbing
- Ledge grabbing
- Air combos
- Dash attacks
- Grappling movement

- Skill trees

# Controls

## Keyboard
| Key | Action |
|---|---|
| WASD | Move |
| Space | Jump |
| Shift | Dodge Roll |
| Ctrl | Sprint |
| Left Click | Attack |
| Right Click | Heavy Attack |
| Q/E/R/F | Skills |
| Tab | Inventory |
| C | Crafting |
| M | Map |
| Esc | Pause |

---

## Controller
| Button | Action |
|---|---|
| Left Stick | Move |
| A / Cross | Jump |
| B / Circle | Dodge |
| RT / R2 | Attack |
| LT / L2 | Heavy Attack |
